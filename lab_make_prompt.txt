첨부한 두 파일을 사용해서 가상실험 HTML을 만들어 줘.

[첨부 파일]
1. 가상실험실_뼈대_채우기_지시서.md — 작업 지시서
2. 가상실험실_뼈대_v2.html — 베이스 HTML 파일

[작업 방법]
1. 가상실험실_뼈대_v2.html을 베이스로 사용해 줘. 처음부터 새로 작성하지 마.
2. 지시서의 ★① ~ ★⑧ 항목에 따라 뼈대 HTML의 해당 마커 영역만 채워 넣어 줘.
3. 마커 밖의 코드(공통 구조, 변수명, 함수명, CSS 클래스명, 드래그 시스템, screenToWorld, BGM/TTS 채널, updateSimulation, resetExperiment 등)는 절대 수정하지 마.
4. var만 사용 (let/const 금지).
5. 결과는 단일 HTML 파일 하나로, 코드 스니펫이 아니라 바로 실행 가능한 완성된 형태로 줘.
6. 결과 파일명: 지시서에 명시된 형식 그대로 따라 줘.

[작업 시 유의사항]
- 지시서에 명시된 부분만 작업하고 나머지는 건드리지 마. 임의로 "더 좋게" 바꾸지 마.
- 음성 파일(mp3)은 지시서의 audioFiles 매핑대로 파일명만 정확히 넣어 줘. 실제 mp3가 없으면 뼈대 코드의 TTS fallback이 자동으로 처리해.
- ★⑦ 시뮬레이션 코어는 가장 중요한 부분이야. 지시서의 시뮬레이션 설명을 정확히 반영해서 작성해 줘.
- 작업 완료 후, 다음을 한 번 더 확인해 줘:
  · 모든 ★ 마커 영역이 빠짐없이 채워졌는지
  · 마커 밖 코드를 수정한 부분이 없는지
  · let/const를 사용한 곳이 없는지
  · 단계 수와 dot 개수, audioFiles 항목 수가 일치하는지
  · 사용자 표시용 Step 번호는 1부터 시작하고, JS 코드 키(step0, take2 등)는 0-base를 그대로 따름 — 둘이 어긋나 보이는 건 의도된 것이니 수정하지 마

[제공 형태]
완성된 HTML 파일을 다운로드 가능한 형태로 제공해 줘.
